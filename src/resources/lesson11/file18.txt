<h2>Why Software Developers Choose Java</h2>
<p>Java has been tested, refined, extended, and proven by a dedicated community of Java developers, architects and enthusiasts. Java is designed to enable development of portable, high-performance applications for the widest range of computing platforms possible. By making applications available across heterogeneous environments, businesses can provide more services and boost end-user productivity, communication, and collaboration&mdash;and dramatically reduce the cost of ownership of both enterprise and consumer applications. Java has become invaluable to developers by enabling them to:</p>

<ul><li>Write software on one platform and run it on virtually any other platform</li>
<li>Create programs that can run within a web browser and access available web services</li>
<li>Develop server-side applications for online forums, stores, polls, HTML forms processing, and more</li>
<li>Combine applications or services using the Java language to create highly customized applications or services</li>
<li>Write powerful and efficient applications for mobile phones, remote processors, microcontrollers, wireless modules, sensors, gateways, consumer products, and practically any other electronic device</li>
</ul>

	<h3>What Is JavaFX</h3>
	<p><strong>JavaFX is powered by Java.</strong> The JavaFX platform enables application developers to easily create and deploy Rich Internet Applications (RIA) that behave consistently across multiple platforms. JavaFX extends the power of Java by allowing developers to use any Java library within JavaFX applications. This way developers can expand their capabilities in Java and make use of the presentation technology that JavaFX provides to build engaging visual experiences.</p>
	<p>To learn more about JavaFX technology, visit <a href="http://www.oracle.com/us/technologies/javafx/overview" title="JavaFX site">JavaFX</a> web site for more information.</p>
